# BusinessStrategyQuest

An interactive learning game to build business strategy skills.  
Players navigate Vision, Strategy, and Action choices to earn points and unlock results.  

**Features:**
- Brand selection: Fetina Bonds / Scent of Beauti
- Score tracking
- Copy-safe educational link: [www.marykay.com/n-2024](https://www.marykay.com/n-2024)
- Playable directly in a browser via Twine HTML file

## How to Play
1. Open `twine/business_strategy_game.html` in your browser.
2. Choose your brand.
3. Make strategic decisions at each level.
4. See your results and score at the end.
